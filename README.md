# Mini Paint em Java
Este projeto consiste no desenvolvimento de uma aplicação gráfica semelhante ao clássico Microsoft Paint, com foco na compreensão prática de conceitos fundamentais de programação gráfica e interação com o mouse.

## Funcionalidades
- 🎨 Escolher a cor do pincel
- 📏 Ajustar o tamanho do pincel
- 🖱️ Desenhar livremente utilizando o mouse

## Como executar
```
git clone https://github.com/dev-pedr0/Java-Paint
Abra o projeto no IntelliJ IDEA
Execute a classe principal (Main.java)
```
